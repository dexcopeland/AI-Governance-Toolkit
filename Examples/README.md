# Worked Examples

Fully filled-in reference artifacts so you and your reviewers can see the **depth and tone** expected. **All fictional — for reference only, not real decisions.**

## The connected scenario: "TalentSift"
A single fictional use case runs through the whole intake pipeline so you can see how the artifacts link:

> **TalentSift** — a third-party (SaaS) generative-AI assistant that ingests job applicants' resumes, scores their fit against a job description, and produces a ranked shortlist with AI-written candidate summaries. Recruiters review the shortlist. ~5,000 applicants/year.

It's deliberately a **high-stakes** case (decisions about people + PII + well-known bias risk) so the examples exercise the hard parts of the rubric.
